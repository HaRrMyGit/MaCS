# Формулировка
$\sqsupset f,g \in R[a,b] \ f > 0, \ m,M \in \mathbb{R} \ m \leq f \leq M$
Тогда $\exists \mu \in [m,M]: \int^b_afg = \mu \int^b_ag$
# Доказательство
$a < b, \ g > 0$
Тогда $\int^b_ag > 0$ и $mg \leq fg \leq Mg \Rightarrow m \int^b_ag \leq \int^b_afg \leq M\int^b_ag$
$\Rightarrow$ Если $\int^b_ag = 0,$ то $\int^b_afg = 0 \Rightarrow$ верно $\forall\mu$
Если $\int^b_ag > 0,$ то положим $\mu = \frac{\int^b_afg}{\int^b_ag} \Rightarrow$ выполнены условия
$$\square$$
# Следствие 1
$\sqsupset f \in C[a,b]$
$g \in R[a,b], \ g \geq 0 (\leq 0)$
Тогда $\exists c\in [a,b]: \int^b_afg = f(c)\int^b_ag$
# Следствие 2
$\sqsupset f \in R[a,b], \ m,M \in \mathbb{R}, m \leq f \leq M$
Тогда $\exists \mu \in [m,M]: \int^b_af = \mu(b-a)$
# Следствие 3
$\sqsupset f \in C[a,b]$
Тогда $\exists c \in [a,b]: \int^b_af = f(c)(b-a)$
$R[a,b]$ - [[Класс интегрируемых на отрезке функций]]
$\int^b_a$ - [[Определённый интеграл]]
