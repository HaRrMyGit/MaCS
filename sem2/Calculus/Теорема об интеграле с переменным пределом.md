# Формулировка
$E$ - невырожденный промежуток, $f: E \rightarrow \mathbb{R}$, $f$ интегрируема на $\forall$отрезке, содержащемся в $E$; $a\in E$, $\Phi(x) = \int^x_af, \ x \in E \Rightarrow$
1. $\Phi \in C(E)$
2. Если, кроме того, $f$ непрерывна в $x_0$, то $\Phi$ дифференцируема в $x_0$ и $\Phi'(x_0) = f(x_0)$
# Доказательство
## 1
Возьмём $\delta > 0: [x_0 - \delta, x_0 + \delta] \cap E = [A,B]$ - невырожденный промежуток
$\Rightarrow f \in R[a,b] \Rightarrow f$ ограничена на $[A,B]: |f| \leq M$
$\sqsupset \Delta x: x_0 + \Delta x \in [A,B]$
По [[Аддитивность интеграла по отрезку||аддитивности интеграла]] $\Phi(x_0 + \Delta x) - \Phi(x_0) = \int^{x_0 + \Delta x}_af - \int^{x_0}_af = \int_{x_0}^{x_0 + \Delta x}f$
$$|\Phi(x_0 + \Delta x) - \Phi(x_0)| = \left|\int_{x_0}^{x_0 + \Delta x}f\right| \leq \left|\int_{x_0}^{x_0 + \Delta x}|f|\right| \leq M|\Delta x| \to 0 \text{ (при }\Delta x \to 0)$$
## 2
Возьмём $\varepsilon > 0$. 
По определению непрерывности $f: \exists\delta > 0 \ \forall t \in E: |t - x_0| < \delta \Rightarrow |f(t) - f(x_0)| < \varepsilon$
$\sqsupset \Delta x: x_0 + \Delta x \in E$ и $|\Delta x| < \delta \Rightarrow$
$$\left|\frac{\Phi(x_0 + \Delta x) - \Phi(x_0)}{\Delta x} - f(x_0)\right| = \left|\frac{1}{\Delta x}\int^{x_0 + \Delta x}_{x_0}f(t)dt - f(x_0) \right| =$$
$$= \left|\frac{1}{\Delta x}\int^{x_0 + \Delta x}_{x_0}(f(t) - f(x_0))dt\right| < \frac{1}{|\Delta x|}\cdot|\Delta x|\cdot\varepsilon = \varepsilon$$
$$\square$$
# Следствие
Функция, непрерывная на промежутке, имеет на нём первообразную

$R[a,b]$ - [[Класс интегрируемых на отрезке функций]]
$\int^b_a$ - [[Определённый интеграл]]
$\Phi$ - [[Интеграл с перменным верхним пределом]]!
