# Формулировка
$a < b, \ f \in R[a,b], \ f \geq 0 \text{ на } [a,b], \ \exists x_0 \in [a,b]: f(x_0) > 0 \ и \ f \text{ непрерывна в } x_0 \Rightarrow \int^b_af > 0$
# Доказательство
$\varepsilon = \frac{f(x_0)}{2} > 0$ По определению непрерывности $f$ в точке $x_0$ $\exists\delta > 0 \ \forall x \in [x_0 - \delta, x_0 + \delta] \cap [a,b] = [\alpha, \beta] \ f(x) > f(x_0) - \varepsilon = \frac{f(x_0)}{2}$
по [[Аддитивность интеграла по отрезку]] $\Rightarrow \int^b_af = \int_a^\alpha f + \int^\beta_\alpha f+ \int^b_\beta f \geq \frac{f(x_0)}{2}(\beta - \alpha) > 0$
$$\square$$
$R[a,b]$ - [[Класс интегрируемых на отрезке функций]]
$\int^b_a$ - [[Определённый интеграл]]
