# Формулировка
$a <b, \ f,g \in R[a,b], \ f \leq g$ на $[a,b] \Rightarrow \int^b_af \leq \int^b_ag$
# Доказательство
$$\sigma_T(f) \leq \sigma_T(g) |_{\lim_{\lambda \rightarrow 0}}$$
$$\square$$
# Следствие
$a <b, f \in R[a,b]$
$$f \leq M \text{ на } [a,b] \Rightarrow \int^b_af \leq M(b-a)$$
$$f \geq m \text{ на } [a,b] \Rightarrow \int^b_af \geq m(b-a)$$
$R[a,b]$ - [[Класс интегрируемых на отрезке функций]]
$\int^b_a$ - [[Определённый интеграл]]
$\sigma_T$ - [[Интегральные суммы]]
