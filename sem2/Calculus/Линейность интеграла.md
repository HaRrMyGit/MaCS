# Формулировка
$f,g \in R[a,b] \ \alpha, \beta \in \mathbb{R} \Rightarrow \int^b_a (\alpha f + \beta g) = \alpha\int^b_af + \beta\int^b_ag$
# Доказательство
$(\alpha f + \beta g) \in R[a,b]$ ([[Арифметические действия над интегрируемыми функциями]])
$$\sigma_T(\alpha f + \beta g) = \alpha\sigma_T(f) + \beta\sigma_T(g) |_{\lim_{\lambda \rightarrow 0}}$$
$$\square$$
$R[a,b]$ - [[Класс интегрируемых на отрезке функций]]
$\int^b_a$ - [[Определённый интеграл]]
$\sigma_T$ - [[Интегральные суммы]]
