# Формулировка
1. $f \in R[a,b], \ [\alpha,\beta] \subset [a,b] \Rightarrow f \in R[\alpha,\beta]$
2. $a < c < b, \ f \in (R[a,c] \cap R[c,b]) \Rightarrow f \in R[a,b]$ 
# Доказательство
## 1
Возьмём $\varepsilon > 0$, подберем $\delta$ из [[Критерий интегрируемости|критерия интегрируемости]] $f$ на $[a,b]: \exists \delta > 0 \ \forall T \ \lambda_T<\delta \ S_T - s_T < \varepsilon$  
Покажем, что такое $\delta$ подходит для критерия интегрируемости $f$ на $[\alpha, \beta]$:
Возьмём [[Дробление|дробление]] $T_0$ отрезка $[a,b] \ \lambda_{T_0}$  $< \delta$
Возьмём какие-нибудь дробления $[a,\alpha] \ и \ [\beta,b]$ ранга $< \delta$
$\Rightarrow$ Получим дробление $T$ отрезка $[a,b]$ ранга $< \delta$
$$
x_0 < ...<x_\mu < .. < x_\nu < ... < x_n
$$
$x_0 = a, x_\mu = \alpha, x_\nu = \beta, x_n = b$
$$
S_{T_0} - s_{T_0} = \sum^{\nu-1}_{k=\mu}w_k(f)\Delta x_k \leq \sum^{n-1}_{k=0}w_k(f)\Delta x_k = S_T - s_T < \varepsilon
$$
## 2
Проверим критерий интегрируемости $f$ на $[a,b]$
Возьмём $\varepsilon > 0$
Считаем, что $f \neq const$, то есть $w(f)_{[a,b]} > 0$
По [[Критерий интегрируемости|критерию интегрируемости]] $f$ на $[a,c]$ и $[c,b]$
$$
\exists \delta_1 > 0: \forall T_1: \lambda_{T_1} < \delta_1 \ S_{T_1} - s_{T_1} < \frac{\varepsilon}{3}
$$
$$
\exists \delta_2 > 0: \forall T_2: \lambda_{T_2} < \delta_2 \ S_{T_2} - s_{T_2} < \frac{\varepsilon}{3}
$$
$T_1, T_2$ - дробления $[a,c]$ и $[c,b]$ соответственно
Возьмём $\delta = min\{\delta_1,\delta_2,\frac{\varepsilon}{3w}\}$ 
$\sqsupset T$ - дробления $[a,b]$ $\lambda_T < \delta$
Точка $c$ может не принадлежать $T$
$\sqsupset c \in (x_\nu,x_{\nu + 1})$ 
Положим $T' = T \cup \{c\}$ 
$$
T_1 = T' \cap [a,c], \ T_2 = T' \cap [c,b]
$$
$$
S_T - s_T = \sum^{n-1}_{k=0}w_k(f)\Delta x_k \leq w_\nu(f)\Delta x_\nu + S_{T_1} - s_{T_1} + S_{T_2} - s_{T_2} < \varepsilon
$$
$$
\square
$$
