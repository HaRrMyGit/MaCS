# Формулировка
$f,g$ дифференцируемы на $[a,b]$, $f',g' \in R[a,b] \Rightarrow \int^b_afg' = fg|^b_a - \int^b_af'g$
# Доказательство
$f,g$ дифференцируемы $\Rightarrow$ непрерывны $\Rightarrow$ интегрируемы $\Rightarrow$ по [[Арифметические действия над интегрируемыми функциями]] $fg',f'g \in R[a,b]$
$$(fg)' = f'g + fg' \in R[a,b]$$
По [[Формула Ньютона-Лейбница]]
$$\int^b_a(fg)' = \int^b_af'g + \int^b_afg' $$
$$\Vert \ \ \ \ \ \\ \ \ \\ \ \ \ \ \\ \ \ \ \ \\ \ \ \ \ \\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ $$
$$fg|^a_b \ \ \\ \ \ \ \ \ \ \ \ \\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \\ \ \  \ \ $$
$$\square$$
$R[a,b]$ - [[Класс интегрируемых на отрезке функций]]
$\int^b_a$ - [[Определённый интеграл]]
