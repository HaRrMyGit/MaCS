1. Если $b < a, \ f \in R[a,b],$ то $\int^b_af = -\int^a_bf$
2. Для вырожденного отрезка считаем, что $\int_b^a f = 0 \ \forall f$
3. Далее через $[a,b]$ обозначается отрезок с концами $a, b,$ но не обязательно $a < b$
$R[a,b]$ - [[Класс интегрируемых на отрезке функций]]
$\int^b_a$ - [[Определённый интеграл]]
