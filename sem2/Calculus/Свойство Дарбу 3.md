# Доказательство
для одного [[Дробление|дробления]] $T$ очевидно $s_T \leq S_T$
$\sqsupset T_1, \ T_2$ - два разбиения, $T = T_1 \cup T_2 \Rightarrow s_{T_1} \leq s_T \leq S_T \leq S_{T_2}$
$$
\square
$$
