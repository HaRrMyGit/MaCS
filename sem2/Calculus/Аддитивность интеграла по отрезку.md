# Формулировка
$a,b,c \in \mathbb{R} \ f \in R[\min\{a,b,c\}, \max\{a,b,c\}] \Rightarrow \int^b_af = \int^c_af + \int^b_cf$
# Доказательство
$\sqsupset a < c < b \ f \in R[a,b] \Rightarrow f \in (R[a,c] \cap R[c,b])$
$\sqsupset (\bar{T}^{(n)}, \bar{\xi}^{(n)})$ и $(\bar{\bar{T}}^{(n)}, \bar{\bar{\xi}}^{(n)})$ - последовательности [[Оснащение дробления|оснащений дроблений]] отрезков $[a,c]$ и $[c,b]$ на $n$ равных частей
$$T^{(n)} = \bar{T}^{(n)} \cup \bar{\bar{T}}^{(n)}, \xi^{(n)} = \bar{\xi}^{(n)} + \bar{\bar{\xi}}^{(n)}$$
$$\sigma_n, \bar{\sigma_n}, \bar{\bar{\sigma}}_n \text{ - соответствующие интегральные суммы, } \sigma_n = \bar{\sigma_n} + \bar{\bar{\sigma}}_n |_{n \rightarrow \infty}$$
Если $a<b<c:$ по доказанному $\int^c_af = \int^b_af + \int^c_bf \Leftrightarrow \int^b_af = \int^c_af - \int^c_bf =\int^c_af + \int^b_cf$
Остальное аналогично
