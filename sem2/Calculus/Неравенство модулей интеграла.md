# Формулировка
$a < b, \ f \in R[a,b] \Rightarrow |\int^b_af| \leq \int^b_a |f|$
# Доказательство
$$-|f| \leq f \leq |f|$$
$$-\int^b_a|f| \leq \int^b_af \leq \int^b_a|f| \Leftrightarrow |\int^b_af| \leq \int^b_a |f|$$
$$\square$$
$R[a,b]$ - [[Класс интегрируемых на отрезке функций]]
$\int^b_a$ - [[Определённый интеграл]]
