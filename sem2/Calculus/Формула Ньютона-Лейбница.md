# Формулировка
$f \in R[a,b], \ F$ - первообразная $f$ на $[a,b]$
Тогда 
$$\int^b_af = F(b) - F(a)$$
# Доказательство
при каждом $n \in \mathbb{N}$ положим $x_k = a + \frac{k(b-a)}{n}$
Тогда 
$$F(b) - F(a) = \sum^{n-1}_{k=0}(F(x_k+1) - F(x_k))$$
по Теореме Лагранжа $\forall k \in [0:n-1] \ \exists \xi_k^{(n)} \in (x_k,x_{k+1}):$
$$F(x_{k+1}) - F(x_k) = F'(\xi_k^{(n)})\Delta x_k = f(\xi_k^{(n)})\Delta x_k$$
В силу [[Предел интегральных сумм|интегрируемости]] $f:$
$$\int^b_af = \lim_{n \to \infty}\sum^{n-1}_{k=0}f(\xi_k^{(n)})\Delta x_k = \lim_{n \to \infty}(F(b) - F(a)) = F(b) - F(a)$$
$$\square$$
$R[a,b]$ - [[Класс интегрируемых на отрезке функций]]
$\int^b_a$ - [[Определённый интеграл]]
$\xi$ - [[Оснащение дробления]]
