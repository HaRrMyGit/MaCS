#Теорема 
# Формулировка
$\forall \mathbb{v} \ \Gamma_\mathbb{v} \geq 0 \text{ и } = 0, \text{ только если } \mathbb{v} \text{ линейно зависим}$
# Доказательство
$\sqsupset \mathbb{v} \text{ - конечный и линейно зависимый} \Rightarrow X = (x_1,...x_n)| x_i \in \mathbb{R}$
$$X\mathbb{v}^T = 0$$
$$X\mathbb{v}^T\mathbb{v} = 0 \Rightarrow XG_\mathbb{v} = 0 \Rightarrow строки \ G_\mathbb{v} \ \text{ линейно зависимы} \Rightarrow \Gamma\mathbb{v} = 0$$
$\sqsupset \mathbb{v}$ линейно независим
$w = <\mathbb{v}>$ и $\mathbb{v}$ - базис $w$
$\sqsupset \mathbb{e}$ - [[Ортонормированность|ортонормированный]] базис $w$ $\sqsupset \mathbb{v} = \mathbb{e}C_{\mathbb{e}\mathbb{v}}$
$$\Gamma_\mathbb{v} = \det G_\mathbb{v} = \det (C^T_{}\mathbb{e}\mathbb{v} G_\mathbb{e} C_{\mathbb{e}\mathbb{v}}) = \det C^T_\mathbb{v} \det C_{\mathbb{e}\mathbb{v}} = (\det C_{\mathbb{e}\mathbb{v}})^2 > 0$$
$$\square$$
# Следствие
Неравенство КБШ
$$\mathbb{v} = (v,w)$$
$$G_\mathbb{v} = \begin{pmatrix}
(v_1,w_1) & (v_1,w_2) \\
(v_2,w_1) & (v_2,w_2) 
\end{pmatrix} = (v,v)(w,w) - (v,w)^2 \geq 0 \Rightarrow ||v||\cdot||w|| \geq |(v,w)|$$
