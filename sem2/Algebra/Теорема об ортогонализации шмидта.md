# Формулировка

# Доказательство
Не умаляя общности $v_i \neq 0 \ \forall i; \ e_1 := \frac{v_1}{||v_1||}$ 
Индукцией по $m$ база есть, переход: $e_{m+1}$
$$
\sqsupset w = v_{m+1} - (v_{m+1},e_1)e_1 - \dots - (v_{m+1},e_m)e_m
$$
Проверим [[Ортогональность|ортогональность]]
$$
(w,e_i) = (v_{m+1}, e_i) - (v_{m+1},e_i) = 0
$$
Случай $w = 0 \Rightarrow v_{m+1} = \sum^m_{i=1}(v_{m+1},e_i)v_{m+1} \Rightarrow <v_1,...,v_{m+1}> \subset <e_1,...,e_m>$
Случай $w \neq 0 \Rightarrow e_{m+1} = \frac{w}{||w||}$ 
$$
\square
$$
