# Определение
Под поверхностью $X$ будем понимать замкнутую поверхность, гомеоморфную одной из следующих фигур:
$$
сфера \ (0\ дыр),\ тор \ (1\ дыра) ,\ бублик\ (2\  дыры),\ связная\  сумма\  торов\ (g\ дыр)
$$
Число $g$ называется родом поверхности $X$
# Определение
Число $\chi(X) = 2 - 2g$ называется эйлеровой характеристикой поверхности $X$ рода $g$
# Теорема
## Формулировка
Пусть на поверхности $X$ рода $g$ начерчен связный псевдограф, имеющий $B$ вершин, $P$ ребер, и разбивающий $X$ на $\Gamma$ областей, [[Параграф Введение|гомеоморфных]] кругу. Тогда
$$
B - P + \Gamma = 2 - 2g
$$
## Доказательство
Для каждого тора $i = 1,2,...,g$ выберем цикл $C_i,$ который не содержит вершин псевдографа (это возможно, так как число вершин конечно). Из условия гомеоморфности кругу всех областей, на которые псевдограф разбивает поверхность, следует, что каждый такой цикл пересекает хотя бы одно ребро псевдографа ( в противном случае нестягиваемый в точку цикл $C_i$ целиком принадлежал одной из $\Gamma$ областей, что означало бы негомеоморфность этой области кругу). Пусть $k_i \geq 1$ - число точек пересечения цикла $C_i$ с ребрами псевдографа. Добавим эти точки к вершинам псевдографа, а соединяющие эти точки дуги добавим к его рёбрам. Для возникшего таким образом нового связного псевдографа оценим число вершин $B'$, число ребер $P'$, а также число областей $\Gamma'$, на которые он разбивает ту же поверхность $X$
$$B' = B + (k_1, ..., k_g)$$
$$P' = P + 2(k_1, ..., k_g)$$
$$\Gamma' = \Gamma + (k_1, ..., k_g)$$
$$\Rightarrow$$
$$B' - P' + \Gamma' = B - P + \Gamma$$
Разрежем теперь поверхность по циклам $c_1,...,c_g,$ удваивая расположенные на них точки пересечения и стягивающие их дуги. Заклеивая возникшие таким образом дырки $c_1',c_1'',...c_g',c_g''$ дисками, получим поверхность рода 0 с начерченным на ней связным псевдографом, имеющим $B'' = B' + (k_1 + ... + k_g)$ вершин, $P'' = P' + (k_1,...,k_g)$ ребер, и разбивающий сферу на $\Gamma'' = \Gamma' + 2g$ областей. По [[Параграф 1|теореме эйлера]] 
$$2 = B'' - P'' + \Gamma'' = B' - P' + \Gamma' + 2g = B - P + \Gamma + 2g$$
$$\square$$
