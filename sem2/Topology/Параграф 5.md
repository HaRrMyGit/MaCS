# Определение
Множество $X$ называется метрическим пространством, если на нём задана метрика, то есть нетривиальная функция $(x,y) \mapsto \rho(x,y) \geq 0$, удовлетворяющая условиям:
1. $\rho(x,y) = 0 \Leftrightarrow x = y$
2. $\rho(x,y) = \rho(y,x) \ \forall x,y \in X$ 
3. $\rho(x,y) \leq \rho(x,z) + \rho(z,y) \ \forall x,y,z \in X$
Примеры (в которых надо проверить три свойства )
4. евклидово точечное пространство
5. неевклидова метрика на А. метрика в данном случае будет "манхеттенским расстоянием" ($|x_i - x_j| + |y_i - y_j| \dots$) 
6. Подмножество метрического пространства с индуцированной метрикой (метрика будет сужением на подмножество) 
# Определение
Открытым шаром в метрическом пространстве $(X, \rho)$ называется множество вида $B_\varepsilon(x) = \{y \in X | \rho(x,y) < \varepsilon\}$.
# Определение
Подмножество $U \subset X$ называется открытым, если $\forall x \in U \ \exists \varepsilon > 0 \ B_\varepsilon(x) \subset U$  
# Утверждение
## Формулировка
Совокупность $\Omega(\rho)$ всех открытых множеств в метрическом пространстве $(X, \rho)$ обладает свойствами:
1. объединение любого семейства множеств принадлежащих совокупности $\Omega(\rho)$ также принадлежит совокупности 
2. пересечение любого конечного семейства множеств принадлежащих совокупности $\Omega(\rho)$ также принадлежит совокупности $\Omega(\rho)$
3. всё множество $X$ принадлежит совокупности $\Omega(\rho)$
